## Run Locally

Clone the project

```bash
  git clone https://github.com/AFZL210/React-Clipboard.git
```

Go to the project directory

```bash
  cd my-clipboard
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
